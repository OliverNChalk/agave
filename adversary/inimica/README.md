# Inimica

Inimica is part of the invalidator project, designed to attack the Solana network.  Inimica
is a collection of attacks that can be performed without being a validator on the network.

Individual attacks are described in the
[`docs/invalidator/inimica.md`](../../docs/invalidator/inimica.md).

"Inimica" [means "foe" in
Latin](https://translate.google.com/?sl=la&tl=en&text=inimica&op=translate).

## Operation

Inimica is designed to be run next to the `invalidator-client`, being part of a suite of end-to-end
adversarial testing of the network.
